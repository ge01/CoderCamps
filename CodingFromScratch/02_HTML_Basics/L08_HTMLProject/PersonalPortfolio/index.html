About Me

Add picture
My name is and I’m DIYer currently coding my way to a new tech career as a Full Stack Software Developer through the Coder Camps online programming bootcamp. I enjoy working with digital electronics like Transistor Transistor Logic (TTLs) and Programmable Logic Devices (PLDs) to make stuff. A while back I took a state diagram for an automated carwash and simulated it using TTLs and a PLD. Next, I’d like to get into assembly for the 8051 microcontrollers. Actually, I think I’ll go with C for the programming of the 8051. I’m thinking that by working with another high-level programming language, the skills I learn will transfer over to front end and back end JavaScript development.

My Background

For the last seven years I’ve travelled around the nation doing contract work for electronic manufacturing companies. Below is a list of some of the cities I’ve worked and lived in.
  • Dallas, Texas
  • Cedar Rapids, Iowa
  • Cleveland, Ohio

Why I Decided to Learn Programming

A few years ago, I had a job working with System Debug Engineers for a microprocessor company. I was blown away by their ability to use low level assembly to troubleshoot hardware. Since then, I’ve been slowly learning programming from various websites. Now I’m committed to collaborating with others through Coder Camps to spring board my career into Full Stack Development.
